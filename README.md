# calculadora-frequencia-cardiaca

   

Projeto desenvolvido em linguagem C com o objetivo de calcular a frequência cardíaca máxima e as zonas de treino utilizando o método de Karvonen.


- Linguagem C
- GCC
- Terminal / Console


- Cálculo da frequência cardíaca máxima (220 - idade)
- Cálculo das zonas de treino:
  - Leve
  - Moderada
  - Aeróbica
  - Anaeróbica
  - Máxima
- Menu interativo
- Validação de dados
- Loop até o usuário encerrar


```bash
gcc src/main.c -o frequencia
./frequencia
